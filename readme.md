요구사항
- 로그인, 로그아웃
  - 세션기반
- 예매 버튼 클릭시 대기열 진입
  - 대기 순번과 대기 시간 노출
  - 일정 주기마다 대기열 상태 조회해서 순번과 시간 갱신
- 입장 가능시 예매 페이지 진입
  - 토큰 발급하여 일정 시간 이상 지날 경우 대기열로 되돌린다
- 공연일시, 좌석 조회
  - 아직 확정되지 않은 좌석 중 사용자가 선점한 좌석은 선택시 메시지 노출
  - 선점 가능한 좌석 선택시 좌석을 선점한다. 일정시간이 지날 경우 선점은 풀린다.
- 예약 버튼을 클릭하면 결제 페이지로 이동한다.
  - 결제에 성공하면 좌석은 확정 상태로 바뀐다.
  - 결제에 실패하면 좌석의 선점은 풀리고 좌석 선택 화면으로 돌아간다.
